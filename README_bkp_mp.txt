# Sistema de Hotspot Automático - WiFi Barato

## Próximos Passos:

1. Importe o arquivo `database.sql` no MySQL/phpMyAdmin
2. Edite o arquivo `config.php` com suas configurações
3. Configure as credenciais do Mercado Pago no banco de dados
4. Configure a conexão com o MikroTik
5. Adicione o logo em `img/wifi-barato-logo.png`
6. Leia o arquivo `INSTALACAO.md` para instruções completas

## Estrutura:
- database.sql - Script de criação do banco
- config.php - Configurações do sistema
- index.php - Página principal
- *.php - Outros arquivos do sistema
